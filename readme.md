# Getting Started
Set all bash scripts to be executable.
```aidl
chmod u+x *.sh
```
To run the script use `simulation.sh`.
```aidl
./Simulation.sh
```
Output CSV's will appear in the `output` directory after simulation has finished.  
To clean the `src` directory use `clean.sh`.
```aidl
./clean.sh
```
